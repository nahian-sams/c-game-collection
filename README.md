# 🎮 C Game Collection

A simple terminal-based game collection built using C.

## 🚀 Features

* Snake Game
* Word Guessing Game
* Rock Paper Scissors
* Number Guessing Game
* Menu-driven interface

## 🛠️ Technologies Used

* C Programming
* Standard Library
* Console-based UI

## ▶️ How to Run

1. Compile the code:

   ```
   gcc main.c -o game
   ```
2. Run the program:

   ```
   ./game
   ```
